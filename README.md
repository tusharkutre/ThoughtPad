# ThoughtPad : a notes/codes saver app

# Tech stack :
- React.js
- Redux toolkit
- TailwindCSS
- LocalStorage

# Features
- search a paste/note
- CRUD operations
- A single paste/note can be viewed
- A paste can be copy to clipboard

# functionalities 
- added redux setup
- added routing setup
- shows real date and time of a single note
- shows Toast notifications
